SafeGuard: Crash Detection & Emergency SOS Assistant

Usage Scenarios & Recommendations

SafeGuard automatically monitors phone usage to detect signs of user inactivity or possible emergencies. Each time the user interacts with the phone (e.g., unlocking, switching apps), the timer resets. You can set different monitoring durations for various daily or high-risk situations:

  • Home alone: Set a period when you are home alone (e.g., during the night). If no phone activity occurs, emergency contacts will be notified automatically.
  • Working alone or in hazardous environments: Set the duration to match your shift or slightly longer. If no phone activity is detected, an emergency alert will be triggered.
  • Outdoor activities like hiking, jogging: Set your expected return time. If the phone remains inactive, SafeGuard will trigger an SOS alert.
  • Risky meetings or social situations: Set a time when you expect to leave. If overdue and inactive, the app sends an alert.
  • Long-hour solo protection: You can set up to 16 hours of continuous monitoring to cover long periods with no supervision.

Crash Detection Usage Tips

  • When to enable: Turn on before leaving and disable upon arrival to save battery.
  • Where to place your phone: Keep it on your body (e.g., in a pocket). Avoid storing it deep inside bags or compartments.
  • Fallback safety: If a crash goes undetected, SafeGuard uses a "stillness score" system. When the phone detects no movement or location change over time, it automatically sends an alert to your emergency contact. This serves as a backup to ensure help even if no impact was detected.
